# .github
UniVPN — современный Telegram-бот, который помогает быстро оформить доступ к стабильному VPN-подключению. Сервис поддерживает промокод START, открывающий 3 дня пробного периода. UniVPN подойдёт пользователям, которым нужны приватность, защищённый трафик и удобное управление подключением прямо внутри Telegram.
